# 🏥 AI Healthcare Diagnosis Assistant

An AI-powered web application that predicts possible diseases based on user-reported symptoms using Machine Learning. The system analyzes symptoms, predicts the most probable disease, displays its description along with recommended precautions, generates a professional medical report, and securely stores patient records in a database for future reference.

> **⚠️ Disclaimer:** This project is developed for educational and research purposes only. It is not intended to replace professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare professional for medical concerns.

# 🧠 How It Works

1. Register the patient by entering personal details.
2. Select the symptoms experienced by the patient.
3. The trained Machine Learning model analyzes the symptom pattern.
4. The system predicts the most probable disease.
5. Disease description and recommended precautions are displayed.
6. Prediction confidence score is calculated.
7. Patient information is securely stored in the database.
8. A professional PDF medical report can be generated for future reference.
